# data-science-learning
My notes, mini projects and my progress in learning of data science

## Repository Structure

```
data-science-learning/
├── notebooks/      # Jupyter notebooks for experiments and tutorials
├── src/           # Source code and scripts
├── data/          # Datasets (raw and processed)
├── reports/       # Generated reports and outputs
├── notes/         # Learning notes and documentation
└── README.md      # This file
```

### Directory Details

- **notebooks/**: Contains Jupyter notebooks for data exploration, analysis, and experiments
- **src/**: Python scripts and modules for data processing, modeling, and utilities
- **data/**: Storage for datasets used in projects (raw and processed data)
- **reports/**: Generated analysis reports, visualizations, and project outputs
- **notes/**: Personal notes, learning resources, and documentation

## Getting Started

This repository serves as a comprehensive resource for my data science learning journey, including hands-on projects, experiments, and notes.
